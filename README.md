# DevOps & Cloud Portfolio

This repository contains multiple sub‑projects demonstrating real‑world DevOps, Cloud, SRE, and Project Management skills.

## 📂 Projects

- **[terraform-aws](./terraform-aws/)** – Provision a VPC, EC2, RDS using Terraform.
- **[ci-cd-pipeline](./ci-cd-pipeline/)** – Node.js app with GitHub Actions CI/CD, Docker build, test, and deploy.
- **[k8s-manifests](./k8s-manifests/)** – Kubernetes deployment, service, and ingress for a microservice.
- **[monitoring](./monitoring/)** – Prometheus + Grafana stack with custom dashboards and alerts.
- **[sre-playbook](./sre-playbook/)** – SLI/SLO templates, incident response runbooks, chaos experiments.
- **[project-management](./project-management/)** – Agile board, risk register, stakeholder communication plan.

## 🚀 Quickstart

Each sub‑folder contains a `README.md` with specific instructions and `make` targets.
