# Stakeholder Communication Plan

## Communication Matrix

| Stakeholder | Information Need | Medium | Frequency | Owner |
|-------------|------------------|--------|-----------|-------|
| Portfolio reviewers (recruiters) | Project status, architecture, pipeline health | GitHub README badges, ARCHITECTURE.md | Real‑time (automated) | DevOps Engineer |
| Hypothetical team members | Daily tasks, blockers, progress | Stand‑up (async Slack/Teams) | Daily | Scrum Master |
| Hypothetical product owner | Feature completion, sprint goals | Sprint review meeting, sprint report | Bi‑weekly | Scrum Master |
| Security team | Secret scanning results, vulnerability alerts | Automated GitHub issue on detection | On event | DevOps Engineer |
